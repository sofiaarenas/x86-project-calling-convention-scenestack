🎬 Scene Stack

Calling conventions — movie set edition

↑
RIP
Push a prop, or hit Call to start a new scene (new stack frame).
Push Add a prop (argument or local variable) to the active scene's stack.
Pop Remove the top prop from the active scene.
Clear Clear all props from the active scene.
Call Save the return address + old frame pointer, then open a new scene (new stack frame).
Ret Close the current scene and return to the previous one — restores old frame.
